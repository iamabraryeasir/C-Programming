# List of Problems:

1. Write a C Program to print the Area of a Square if a side is given.
2. Write a C Program to print the Area of a Circle if Radius is given.
3. Write a C Program to calculate perimeter of rectangle. Take sides, a & b, from the user.
4. Take a number(n) from user & output its cube(n*n*n).
5. Write comments for programs a & b.
6. Create a program to input name of the person and respond with "Welcome NAME to C World!!"
7. Create a program that declares one variable of each of the fundamental data types (int, float, double, char) and prints their size using sizeof()operator.
8. Fahrenheit to Celsius conversion.
9. Create a program to swap two numbers.
